# GEO 分析プロンプト(LLM 拡張用)

本ツールのスコアはルールベースで再現性を担保している。LLM は
「なぜ引用されにくいか」の定性補足にのみ使う(スコア自体は変えない)。

```
あなたは生成AI検索(ChatGPT/Perplexity)の引用挙動に詳しいGEOアナリストです。
以下のコンテンツについて、生成AIが回答生成時に引用・要約しやすいかを評価してください。

# 評価観点(参考: 本ツールの構造スコア因子)
- 冒頭に結論/直接回答があるか
- 見出しが疑問形で、問いに直接対応しているか
- 箇条書き・表で要点が抽出しやすいか
- FAQ / Q&A 構造があるか
- 統計・数値と出典が明記されているか(最重要)
- 公開日/更新日など鮮度シグナルがあるか
- 「XとはYである」形式の定義文があるか

# 出力
1. 引用されやすさ(高/中/低)とその理由
2. 生成AIが引用しそうな一節を1つ抜粋
3. 引用率を上げる具体的な書き換え案を3つ

# コンテンツ
{{content}}
```
